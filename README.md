## Java 文件与IO流
- `File` 代表文件 文件或目录本身，可以获取文件或目录本身的信息，但是不能访问文件内容 
- `AccessRandomFile` 提供了随机读取或写入文件内容的能力（通过文件指针）

- IO流
   - 字符流
   - 字节流
   
   - 输入流
   - 输出流
